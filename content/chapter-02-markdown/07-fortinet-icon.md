## Fortinet Icons

{{% notice info %}}
If your local preview runs into error with this feature, follow the two steps
outlined in section [Upgrade](../chapter-01-start/05-upgrade.html) in chapter 1.
{{% /notice %}}

To add a Fortinet Icon, use the `{{%/*ficon*/%}}` shortcode:

### Usages

```text

This is a Cloud Icon: {{%/* ficon icon="Cloud" */%}}
```

**Rendered output:**

This is a Cloud Icon: {{% ficon icon="Cloud" %}}

#### Icon size

Use `size` to specify the size of the icon:

```text
This icon is 128px : {{%/* ficon icon="FortiGate" size="128px" */%}}
```

This icon is 128px : {{% ficon icon="FortiGate" size="128px" %}}

#### inline vs block display

By default, icons are displayed inline with text. Use `class="block"`
to put the icon in its own paragraph and centered.

```text
This icon is display by itself (block) and is 128px : {{%/* ficon icon="FortiGate" size="128px" class="block" */%}}
```

This icon is display by itself (block) and is 128px : {{% ficon icon="FortiGate" size="128px" class="block" %}}
