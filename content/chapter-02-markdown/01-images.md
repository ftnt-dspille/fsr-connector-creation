The basic image syntax is

```markdown
![IMG Alt Text](image file name)
```

Example:

```markdown
![Fortinet Logo](fortinet-logo.png)
```

![Fortinet Logo](fortinet-logo.png)

### Inline image

By default, all images are of the page width and displayed as a block.

You can add parameters to change the image to be displayed inline:

```markdown
This is an inline image: ![Fortinet Logo](fortinet-logo.png?classes=inline&height=12px)
```

This is an inline image: ![Fortinet Logo](fortinet-logo.png?classes=inline&height=12px)

### Size of image

You can add parameters to change the display size of the image:

```markdown
![Fortinet Logo](fortinet-logo.png?height=32px)
```

![Fortinet Logo](fortinet-logo.png?height=32px)
