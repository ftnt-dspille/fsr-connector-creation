## Workshop Overview

Welcome to the FortiSOAR Connector Building Workshop! This hands-on workshop will guide you through the process of creating custom connectors for FortiSOAR.

**Duration:**  TODO

**Difficulty:** Intermediate

**What You'll Learn:**

- Core Python concepts needed for connector development
- FortiSOAR connector architecture and components
- How to build connectors using different methods (inside FortiSOAR and through the Pycharm RDK Plugin)
- Testing and debugging your connectors
- Best practices

## Prerequisites

Before starting this workshop, you should have:

- Access to a FortiSOAR instance (version 7.6.0 or later)
- Basic understanding of REST APIs and JSON structure.
- Familiarity with command-line interfaces
- Pycharm 
- Python 3.9.12 or later. (This is the version used in FortiSOAR 7.6.5. Later versions may work, as long as you don't use certain features that are not available in earlier versions. Common example is with later built-in types not present in 3.9.12)
